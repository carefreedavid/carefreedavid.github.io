Step 2: Review

Group 1: Squarespace

Layout

  • Instantly when landing on the page you see that the layout is clean. In this cleanliness you can very quickly learn what the site is about and can be convinced of its content.
  • I feel as if the site is using the “divine rule” to define is landing page’s layout. The focal point is clean image of a laptop on the right of the page. Which seems to be situated in position that agrees with the ratio set out by the golden rule.
  • This has resulted in a lot of open space. But you do not feel as if this is a really effective use of empty space to ensure the ficus is on the beautiful imagery.
  • layout is completely responsive.

Typography

  • They use a very simple san serif font. This allows for the focus to be on the layout of the site. Which in the case of squarespace is what is what they want the user to focus on. The font size is small. Because again the focal point is the design and layout of the site.

Navigation

  • The navigation bar is very simple. With large spacing between the navigation links. I feel this is really effective because smaller spaces would have made the site feel as if there was too much wasted space. So it almost counter balances the lack of content in the body

Overall Flow

  • The slide layout of the site allows for a very very smooth flow. Scrolling down takes you to the next point of interest. This allows for the focus to be completely on what each div/section is focusing on. Allowing very clear communication of content. The only critique of the flow is that when navigating down the slides there is no way to quickly navigate to certain slides.

Group 2: mimarch.net

Layout

  • The landing page of this site takes you to a page split into thirds. It is made up into two equally sized “div” like containers framing a middle section. But after inspecting the element it is actually not seperated into seperate divs, rather in a single div with a row style formatting. Anyway, The middle section contains two images. Which extends above the other two rows. This looks messy and breaks the design of the page. There is no clear focal point of the site. The three rows are fairly distracting and I find it pretty difficult to focus on any particular point which is disorientating. I would center the imagery and have the different sections displayed on

Typography

  • The site again uses the popular San Serif for its typography. I feel personally that the selection of font-family may have not been thought out too much. The font has no character. The curvature of a serif font may have added making the font more serious and even more legible.

Navigation

  • The navigation bar is tiny and in the top right corner. They could have benefitted from centering the nav. This would have shown off more what the site has to offer. I feel now I need to search to know what the site is offering. And that is counter productive

Overall Flow

  • The overall flow of the site is poor. After the rows you scroll down to poorly organized slides. These are stacked like a table. Each slide’s layout is poor and this makes the overall flow feel disjointed and disorganized. Somewhat chaotic. An eas fix for this would be to expand each slide and scrolling down takes you to the next slide. Allowing the user to focus on a single slide at a time. This would generate flow as the user focusses on each section one at a time.
